Task 2: Fine-Tuning Pretrained Models

30 Points

Leverage transfer learning from ImageNet

Objective

Fine-tune a model pretrained on ImageNet to your wildlife dataset

ImageNet contains 1.2M images across 1000 classes‚Äîcan this knowledge transfer to wildlife?

Transfer Learning Concept

What Transfers?

Early layers:

  • Edge detectors
  • Texture patterns
  • Color filters

Later layers:

  • Object parts
  • Compositional features
  • High-level patterns

The Strategy

  1. Load pretrained weights
  2. Replace final layer for your classes
  3. Freeze early layers (optional)
  4. Fine-tune on your data

Lower learning rate (10-100√ó smaller) prevents destroying pretrained features

Layer Freezing Strategies

Strategy When to Use Pros Cons
Freeze all but final layer Very small dataset (<1000 images) Fast, prevents overfitting Limited adaptation
Freeze early layers Medium dataset (1K-10K images) Good balance Need to choose cutoff
Fine-tune all layers Large dataset (>10K images) Maximum adaptation Slower, risk overfitting

Experiment! Try different strategies and document which works best for your dataset

Deliverables for Task 2

  1. Fine-tuned model with documented layer freezing strategy
  2. Training curves comparing convergence to Task 1
  3. Final test accuracy
  4. Analysis:
    • Which layers were frozen and why?
    • How much faster did it converge?
    • How does accuracy compare to training from scratch?
  5. Training time comparison with Task 1

Task 3: Zero-Shot Classification

20 Points

Use SigLIP 2 vision-language model

Objective

Classify images without any training on your dataset

Critical: Zero-shot means NO training on your dataset‚Äîonly evaluation!

What is SigLIP 2?

Vision-Language Models

Models that understand both images and text

  • Trained on millions of image-text pairs
  • Learn to match images with descriptions
  • Can classify without task-specific training

How It Works

  1. Encode image ‚Üí vector
  2. Encode text prompts ‚Üí vectors
  3. Compute similarity scores
  4. Choose highest score

All classification happens via text prompts!

Prompt Engineering

Simple Prompts

"dog"
"cat"
"elephant"

Better Prompts

"a photo of a dog"
"a photo of a cat"
"a photo of an elephant"

Advanced: Multiple Prompts

For "dog":

"a photo of a dog"
"a dog in nature"
"a picture of a dog"

‚Üí Average similarities

Experiment! Document which prompts work best

Optional: Fine-Tuning on Embeddings

Alternative approach: Train a classifier on SigLIP 2 embeddings

Method

  1. Extract image embeddings
  2. Train logistic regression
  3. Evaluate on test set

Benefits

  • Very data-efficient
  • Fast training
  • Shows transfer learning power

If you do this, include it in your learning curves!

Analysis Questions

Your report should address these key questions:

1. Data Efficiency: Which approach requires the least data to achieve good performance?
2. Crossover Point: At what data size does training from scratch match fine-tuning?
3. Zero-Shot Competitiveness: In which data regime (low/medium/high) does zero-shot perform best relatively?
4. Computational Cost: How does training time scale with data size? Which approach is most efficient?
